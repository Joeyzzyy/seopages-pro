## ğŸ¨ Quality Standards for Every Page

### Required Sections (10 components)

1. **Hero Section**
   - Attention-grabbing headline
   - One-sentence value proposition
   - Above-fold CTA

2. **Quick Verdict Box**
   - 30-second decision summary
   - Best for X, best for Y
   - Star ratings

3. **Feature Comparison Table**
   - Side-by-side features
   - Check/X icons
   - Highlighting differences

4. **Pricing Comparison**
   - Transparent pricing tiers
   - Value analysis
   - Hidden costs exposed

5. **Pros & Cons**
   - Both products fairly evaluated
   - Specific, not generic

6. **Use Case Scenarios**
   - Who should use Product A
   - Who should use Product B
   - Migration considerations

7. **User Reviews Summary**
   - G2/Capterra aggregated
   - Real user quotes
   - Sentiment analysis

8. **Migration Guide**
   - Step-by-step switching
   - Data export/import
   - Timeline estimates

9. **FAQ Section**
   - 5-10 common questions
   - Schema markup ready
   - Voice search optimized

10. **Strong CTA Section**
    - Clear next action
    - Limited time offers
    - Social proof reminder

## ğŸ”§ Technical SEO Requirements

### Per-Page Checklist

- [ ] Title tag: `[ProductA] vs [ProductB]: Best Alternative 2026`
- [ ] Meta description: 155-160 characters with CTA
- [ ] H1: Primary keyword included
- [ ] H2s: Secondary keywords
- [ ] Image alt tags: Descriptive
- [ ] Internal links: 3-5 to related pages
- [ ] External links: 1-2 to authoritative sources
- [ ] Schema markup: FAQ, Product, Review
- [ ] Mobile responsive: All breakpoints
- [ ] Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1
- [ ] Canonical URL set
- [ ] Open Graph tags
- [ ] Twitter Card tags
